# 📜 The TORIS Manifesto: Beyond the Vector

## The Failure of the Vector Space
For a decade, we have tried to represent "meaning" as a coordinate in a high-dimensional space. But meaning is not a location; meaning is a **relationship**. When we force a concept into a vector, we lose the *type* of the connection. A "Cat" is not a point; it is a node in a web of causal, conditional, and contradictory relations.

## The PTP Breakthrough: Goal-Driven Hardware
The most significant bottleneck in local AI is the movement of data between RAM and VRAM. Traditional models page "layers"—large, dense blocks of weights. 

**TORIS introduces Predictive Topological Paging (PTP).**

Because TORIS is a sparse graph, we can treat memory as a topological map. The **Goal Manifold** acts as a compass. If the AI is reasoning about "Quantum Physics," the Goal Manifold warps the field, highlighting the "Quantum" cluster. PTP then pages only that specific topological neighborhood into the GPU.

This means the hardware is no longer a passive container for weights; it becomes a dynamic extension of the reasoning process. **We are not just loading data; we are warping the physical memory layout to match the conceptual goal.**

## The End of Backpropagation
Backpropagation is a global operation—it tries to fix everything at once. TORIS uses **Surprise Gradients**. When the system predicts a relation and observes a different one, it generates a local $\Delta S$ (Surprise) signal. Only the surprised parts of the network compute. This is the mathematical realization of *attention* without the $O(n^2)$ cost of the attention mechanism.
